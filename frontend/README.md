# Ecommarce

This repository contains the Angular front-end and a .NET 8 backend API.

## Structure

- `src/` - Angular application
- `backend/Ecommarce.Api/` - .NET 8 API (authentication + dashboard management)

See `backend/README.md` for API setup and configuration details.

## Environment configuration

Update `src/environments/environment.ts` to change the base API URL used by all
frontend services.
